Micronaut Framework

Que) What is micronaut Framework ?
It is a JVM-based full stack microservice Framework.
Easily testable microservice  Application.


Que) Advantage of micronaut Framework ?
Fast startup time
Ease unit Test
Reduced memory footprint

Que) Different Between Spring boot and Micronaut ?

Application.properties   ===>  application.yml
@Restcontroller ====> @Controller
@RestControllerAdvice  =====>   @Singleton, @Requires
@Service ===> @Singleton
JpaRepository ====>    CrudRepository
@RequestBody ====> @Body
@PostMapping =====> @Post
@Service ====>  @Singleton
@Repository-jpa ===> @Repository-micronaut
Exception Handling  Different to Spring boot
@RequestParam ====> @QueryValue
